O que é Jquery

jQuery é uma biblioteca JavaScript desenvolvida para simplificar os scripts client side que interagem com o HTML.

Usada por cerca de 41% dos 10 mil sites mais visitados do mundo, jQuery é a mais popular das bibliotecas JavaScript de código aberto.

Funcionalidades

  • Resolução da incompatibilidade entre os navegadores.
  • Redução e reutilização de código.
  • Utilização de uma vasta quantidade de plugins criados por outros desenvolvedores.
  • Trabalha com AJAX e DOM.
  • Implementação segura de recursos do CSS1, CSS2 e CSS3.
